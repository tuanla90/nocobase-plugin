# @ptdl plugin registry

Catalog of the `@ptdl` NocoBase plugins in this workspace.
Source: `packages/@ptdl/*`. Packaged tgz: `latest/@ptdl/` (current) + `archive/@ptdl/` (history).
**24 active plugins.** Last updated: 2026-07-16.

> **Bilingual UI i18n — DONE for all plugins (2026-07-15).** Every plugin's *internal* UI strings (not just the manifest `displayName`) now route through NocoBase i18n and switch en/vi. Scheme by source language: VN-source plugins use the Vietnamese string as the i18n key + `en-US.json` (safe — nb i18next has no `fallbackLng`, so a vi-VN miss returns the key; see `docs/SHARED-DEDUP-AUDIT.md` sibling note & the build-guide §R1); English-source plugins (change-log, global-search, status-flow, custom-icons, pwa) use English keys + `vi-VN.json`. All 12 touched this session built, deployed to `nb-local`, and verified live (served bundle `addResources` + inlined locale). **Known remaining gap:** `@ptdl/shared/fieldPicker.tsx` hardcodes `＋ Chèn cột` / `Chọn field…` in Vietnamese → leaks VN into every plugin's field picker even for en users; fix belongs in shared.
